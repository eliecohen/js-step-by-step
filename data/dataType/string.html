The String Type

  • The string type is used to represent text. Each character is a 16-bit values and therefor could be used to represent a unicode character.
  • JavaScript strings are immutable: Once a string is defined it cannot be changed.
  • Strings use zero-based indexing, its length correspond to the number of element in it.
  • Empty string have a length of 0 and is evaluating to false


var foo = "";         //foo is defined.
foo = 10;        //foo is now a number

var bar = 10;     //foo

            

var foo;         //foo is defined, but its type is still undefined
foo = 10;        //foo is now a number
foo = "abc";     //foo is now a string
foo = true;      //foo is now a boolean

            

// the boolean type
foo = true;        //foo is a boolean with true
foo = false;     //foo is a boolean with a false


foo = "abc";     //foo is now a string
