Adding Elements

  • To add a new element to the end of an array you can use the push method.
  • To remove the last element of an array you can use the pop method
  • you can explicitly insert an element at a given position using ...bracket with index (improve)
  • To delete an element, delete operation (to improve)


//The push method

var color = ['yellow','blue'];
color.push('orange');       //adding a new element at the end
console.log(color);         //["yellow", "blue", "orange"]

//The index method

var color = ['yellow','blue'];
color[2]= 'orange';         //adding a new element at index 2
console.log(color);         //["yellow", "blue", "orange"]



            


var color = ['blue','yellow'];
color.pop();
console.log (color); //["blue"]
