Array Introduction

  • An array is an ordered collection of elements. An element can contain the value of any javascript elementary type such as string, object or even be an other arrays. An Array csn contain elements of different type.
  • JavaScript arrays are dynamic and can be changed at runtime. An Element is represented by its position (or index). The first element in the array has an index of 0, the second element an index of 1 and so on.
  • Every JavaScript array has a length property. For nonsparse arrays, this property specifies the number of elements in the array. For sparse arrays, length is larger than the index of all elements.(copy)
