Array Creation

  • Array can be created either using the object literal notation or using its constructor function
  • You read an array element using the [] operators with it position.
  • Add element to array ....


//create an empty array

var color = [];          //array literal method
var color = new Array(); //constructor method

//create a non empty array

var color = ['blue','yellow'];            //array literal method
var color = new Array ('blue','yellow');  //constructor method



            


var color = ['blue','yellow'];

// print the first element of the array

console.log (color[0]); //blue
