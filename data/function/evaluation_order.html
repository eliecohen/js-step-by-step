Function Declaration & Expression Evaluation Order

  • Function expressions are evaluated in their respective place. Thus calling a function before it is declared will throw an error
  • Function declarations are always evaluated before the rest of the code. Thus the location of the call has n....
  • Named function is a special type of function expression. Instead of an anonymous function a name one is used. named function expressions are useful for debugging and profiling
