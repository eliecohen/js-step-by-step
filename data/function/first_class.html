First Class: Function as Value

  • As seen for example in the function expression method, variable can contain function definition. This property is known as first class variable.
  • Like any other variable, function be pass as argument to function or be return by a function.
  • Passing a anonymous callback function as argument to a function is a well known pattern in javascript
