Function Declaration & Expression

  • Function can be defined using either using 1) the function declaration method or 2) the function expression method. Both methods behave exactly the same in their invokation process. They differ in their syntax definition and their evaluation order
  • A function declaration creates (sous...) a new variable using the name of the function and store the new function to it.
  • A function expression is always place a droite ....and return a reference to the function
  • Named function expression are a special type of function expression. Instead of using an anonymous function a name function is used
  • Named function expressions are useful for debugging/profiling application.
