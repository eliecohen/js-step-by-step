Scope: Variable Scope

  • A variable can be defined in either local or global scope and will live throughout runtime and can be accessed and altered in any scope.
  • Local variables exist only within the function body of which they are defined (pas if)
  • A variable declared outside of a function body is a global global variable
  • a variable declared inside of a function without var is global variable
