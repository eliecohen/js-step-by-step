Scope: Variable Scope

  • The scope of a variable is controlled by its declaration location. JavaScript has two type of scopes, global and local
  • Global variables are variables declared outside a function definition. They are accessible and can be altered everywhere in the code. In a browser context thay are attached to the window object. in the nodejs context to the ......
  • Local variables exist only within the function body of which they are defined
  • unlike many other languages, Javascript does not support block level scoping
  • Variable should always be declared with the var keyword, Otherwise transform to global one. To avoid error strict it is best practice to executed in "strict mode" (by adding the "strict mode" in ) With strict mode, you c (move to a new page?)
